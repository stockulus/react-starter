# Small react base Project

Contains just a gulp build process, for a pure client side base React.js Project
Including React Testing
Made for Easy Development

## based on

- react
- redux
- gulp
- watchify
- babel
- brwoser-sync

## How to use

- copy / clone
- npm install
- npm start
- check out other npm run scripts
